# 要件定義書

## 1. 目的

ユーザーが指定したポッドキャストのRSSフィードから、聞きたいエピソードの音声データを簡単にダウンロードできるコマンドラインアプリケーションを提供する。

## 2. 機能要件

### 2.1. 入力

- ダウンロード対象のポッドキャストのRSSフィードURLを、コマンドラインの引数として受け取ること。

### 2.2. 処理

1.  **RSSフィードの解析:**
    -   指定されたRSSフィードURLからフィード情報を取得し、解析する。
2.  **エピソード一覧の表示:**
    -   解析したフィードからエピソードのリスト（タイトル、公開日など）を抽出し、ユーザーが選択しやすいように番号付きでコンソールに表示する。
3.  **エピソードの選択:**
    -   ユーザーにダウンロードしたいエピソードの番号を入力させる。
    -   複数の番号を指定することで、複数のエピソードを一度に選択できること（オプション）。
4.  **音声ファイルのダウンロード:**
    -   選択されたエピソードに対応する音声ファイル（MP3など）をダウンロードする。
5.  **進捗表示:**
    -   ダウンロードの状況（開始、完了など）がわかるメッセージをコンソールに表示する。

### 2.3. 出力

-   ダウンロードした音声ファイルを、実行ディレクトリ配下の `downloads` フォルダに保存する。
-   ファイル名は、エピソードのタイトルを元にした分かりやすい名前（例: `エピソード名.mp3`）にすること。

## 3. 非機能要件

### 3.1. 実行環境

-   Python 3がインストールされている環境のコマンドラインで動作すること。

### 3.2. 使用ライブラリ

-   `feedparser`: RSSフィードの解析に使用する。
-   `requests`: 音声ファイルのダウンロードに使用する。

### 3.3. エラーハンドリング

-   無効なRSSフィードURLが指定された場合に、エラーメッセージを表示して終了すること。
-   ネットワーク接続がない、または不安定な場合に、エラーメッセージを表示すること。
-   ファイルの書き込みに失敗した場合に、エラーメッセージを表示すること。
