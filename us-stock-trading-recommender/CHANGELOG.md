# 変更履歴

## Version 2.1.0 - Python 3.10+対応と最新化 (2025-01-XX)

### 更新
- **Python 3.10+対応**: 型ヒントを最新のPython構文に更新
  - `typing.List` → `list`
  - `typing.Dict` → `dict`
  - `typing.Optional[T]` → `T | None`
  - Python 3.10以降の組み込み型ヒントを使用（推奨: Python 3.12+）
- **依存ライブラリの最新化**:
  - `yfinance`: 0.2.28 → 0.2.45
  - `gspread`: 5.12.0 → 6.1.0
  - `google-auth`: 2.23.0 → 2.35.0
  - `google-auth-oauthlib`: 1.1.0 → 1.2.1
  - `google-auth-httplib2`: 0.1.1 → 0.2.0
  - `pandas`: 2.0.0 → 2.2.0
  - `numpy`: 1.24.0 → 1.26.0
  - `python-dotenv`: 1.0.0 → 1.0.1
- **ドキュメント更新**:
  - `README.md`: Python 3.10+要件を明記
  - `SETUP.md`: Python 3.10+要件を明記

### 修正
- 非推奨となった`typing`モジュールの使用を削除
- すべての型ヒントを最新のPython構文に統一
- コードの互換性と保守性を向上

## Version 2.0.0 - 投資哲学統合版

### 新機能

#### 投資哲学統合分析
- **ベンジャミン・グレアムのバリュー投資**: 安全余裕（マージン・オブ・セーフティ）の計算、P/E、P/B、ROEによる割安判定
- **ウォーレン・バフェットの長期投資**: 優良企業の選別、長期保有推奨、キャッシュフロー重視
- **ウィリアム・J・オニールのCAN SLIM**: 四半期利益成長、年間利益成長、新高値検出
- **広瀬隆雄のプロトコル**: 営業CFマージン15%以上、過去3年の成長率評価、粉飾リスク検出

#### ファンダメンタル分析機能
- 財務指標の取得（P/E、P/B、ROE、EPS、キャッシュフロー等）
- 内在価値の計算（PER法、PBR法、DCF法）
- 安全余裕の計算
- 過去3年の成長率分析

#### 投資哲学レポート
- 各投資家の視点からの詳細レポート生成
- 統合判定と賢人のアドバイス表示
- `--philosophy-report` コマンドで個別銘柄の詳細分析

#### 通知機能の強化
- 売買推奨に賢人のアドバイスを追加表示
- ファンダメンタル分析結果の統合

### 改善点
- テクニカル分析とファンダメンタル分析の統合
- 長期投資視点の追加
- より包括的な投資判断の支援

### 技術的変更
- `fundamental_analyzer.py`: 新規追加
- `investment_philosophy.py`: 新規追加
- `philosophy_report.py`: 新規追加
- `trading_signal.py`: ファンダメンタル分析統合
- `notification.py`: 賢人のアドバイス表示追加
- `main.py`: 投資哲学レポート生成機能追加
