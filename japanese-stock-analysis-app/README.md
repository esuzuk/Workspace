# 日本株分析・バックテストアプリ

Yahooファイナンスから日本株のデータを取得し、バックテストと戦略最適化を行うアプリケーションです。

## 機能

- 📊 **データ取得**: Yahooファイナンスから日本株のデータを自動取得
- 📈 **テクニカル指標**: 移動平均線、RSI、MACD、ボリンジャーバンドなどの指標を自動計算
- 🔍 **バックテスト**: 様々な戦略を過去データで検証
- ⚙️ **戦略最適化**: パラメータを自動最適化して利益が出る条件を探索
- 📋 **利益条件分析**: どのような条件で利益が出るかを統計的に分析

## インストール

```bash
# リポジトリをクローン（またはこのディレクトリに移動）
cd japanese-stock-analysis-app

# 依存関係をインストール
pip install -r requirements.txt
```

## 使い方

### Streamlitアプリの起動

```bash
streamlit run app.py
```

ブラウザが自動的に開き、アプリが表示されます。

### コマンドラインでの使用例

```bash
# 使用例スクリプトを実行
python3 example_usage.py
```

このスクリプトでは、基本的なバックテストの実行例が含まれています。

### 基本的な使い方

1. **データ表示タブ**
   - ティッカーシンボルを入力（例: 7203 = トヨタ自動車）
   - 期間を選択
   - 株価チャートとテクニカル指標を確認

2. **バックテストタブ**
   - 戦略タイプを選択（移動平均クロス、RSI戦略、MACD戦略）
   - パラメータを設定
   - 「バックテスト実行」をクリック
   - 結果を確認

3. **戦略最適化タブ**
   - 最適化タイプを選択
   - 「最適化実行」をクリック
   - 利益が出る戦略のリストを確認

4. **利益条件分析タブ**
   - 最適化実行後、このタブで詳細な分析を確認
   - 最良の戦略とパラメータ範囲を確認

## 日本株のティッカーシンボル例

- `7203` - トヨタ自動車
- `6758` - ソニーグループ
- `9984` - ソフトバンクグループ
- `9434` - ソフトバンク
- `4063` - 信越化学工業
- `6861` - キーエンス
- `8035` - 東京エレクトロン

## 主要な戦略

### 移動平均クロス戦略
短期移動平均が長期移動平均を上抜けしたら買い、下抜けしたら売り

### RSI戦略
RSIが低い水準（例: 30以下）で買い、高い水準（例: 70以上）で売り

### MACD戦略
MACDがシグナルを上抜けしたら買い、下抜けしたら売り
