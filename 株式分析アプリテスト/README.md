# 米国株式売買タイミング推奨アプリケーション

CrewAIを使用して、複数のAIエージェントが協力して米国株式の売買タイミングを分析・推奨するアプリケーションです。

## 機能

- **テクニカル分析**: 価格チャート、テクニカル指標、トレンド分析
- **ファンダメンタル分析**: 財務状況、業績、企業評価
- **統合推奨**: テクニカルとファンダメンタルの分析を統合した売買戦略
- **レポート生成**: 分析結果をMarkdown形式のレポートとして出力

## 必要な環境

- Python 3.8以上
- OpenAI API Key

## セットアップ

1. 依存関係をインストール:
```bash
pip install -r requirements.txt
```

2. 環境変数を設定:
`.env.example`を`.env`にコピーし、OpenAI API Keyを設定してください。

```bash
cp .env.example .env
# .envファイルを編集してOPENAI_API_KEYを設定
```

## 使用方法

アプリケーションを起動:
```bash
python main.py
```

起動後、分析したい株式シンボル（例: AAPL, MSFT, GOOGL）を入力してください。

## アーキテクチャ

このアプリケーションは以下の4つのエージェントで構成されています:

1. **テクニカルアナリスト**: 価格チャートとテクニカル指標を分析
2. **ファンダメンタルアナリスト**: 企業の財務状況と業績を分析
3. **トレーディングアドバイザー**: 両方の分析を統合して売買戦略を推奨
4. **レポートライター**: 分析結果を構造化されたレポートとしてまとめる

## 出力

分析結果は以下の形式で出力されます:

- コンソールに表示
- `reports/`ディレクトリにMarkdownファイルとして保存
